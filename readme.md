## ✨ Projetos em Destaque

Confira alguns dos trabalhos que refletem minha paixão e minhas habilidades:

### **Florinda Eats** (2026)

Um sistema de cardápio digital completo, focado em comunicação em rede, persistência de dados e as melhores práticas de Java moderno. Este projeto demonstra minha capacidade de construir soluções Back-end complexas e integradas.

*Tecnologias:* Java, Gradle, MySQL, Docker, HTTP, Serialização.

### **Acolhe Pet** (2026)

Um sistema de cadastro de pets via CLI, desenvolvido em Java com foco em Programação Orientada a Objetos (POO), persistência em arquivos e validações de domínio. Um exemplo da minha habilidade em criar aplicações robustas e bem estruturadas.

*Tecnologias:* Java, Maven, CRUD, Manipulação de Arquivos.

### **Voluntari-ei** (2025)

Meu projeto de TCC: um aplicativo móvel inovador que conecta instituições filantrópicas a voluntários. Este trabalho destaca minha experiência com desenvolvimento mobile e a criação de soluções com impacto social.

*Tecnologias:* React Native, Prisma ORM, Expo, PostgreSQL, Fastify.
